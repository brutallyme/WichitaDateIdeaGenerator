# WichitaDateIdeaGenerator
Date ideas for Wichita, KS with a local restaurant, fashion style, and after meal activity.

A random number function will be created to pick one option from each of the three arrays: restaurant, dress, activity. When the button is clicked it will randomly select and display the selected option from the array to give a date idea for those local to Wichita, KS and that have a hard time coming up with ideas for dates or would just like to try something different.
